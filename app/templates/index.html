Что такое Post Poster?

Сервис для автоматизации поиска контента для сообществ в социальных сетях, настройки отложенных публикаций и анализа аудитории.

Просто ищет нужный контент в Интернете и публикует его в вашем сообществе.

Сегодня сервис находится на стадии разработки и тестируется на сообществе для iOS разработчиков - iOStory. Бот ищет статьи, новости и другие публикации по тематическим сайтам и публикует их в группе.

Технологии

Post Poster Bot - веб-сервис, в основе которого лежит микрофреймворк Flask, который позволяет быстро развернуть приложение онлайн.

Поиск контента по сайтам осуществляется с помощью Python-скрипта, который при помощи библиотеки Beautiful Soup собирает нужную информацию с выбранных ресурсов.

Для хранения собранного контента используются базы данных PostgreSQL.

Для безопасной авторизации пользователей (требуемой для CMS) используется библиотека Flask-Security.

Публикация и анализ сообществ в VK требует доступа к группам, который осуществляется через VK API. Для быстрой работы с API используется библиотека vk.

Поиск контента и публикация записей происходит постоянно, что требует непрерывной работы скриптов поиска и публикации. При этом, клиентская часть сервиса должна также находиться в рабочем состоянии. Для решения задачи параллельного исполнения использована библиотека Redis Queue.

Веб-приложение работает на облачной платформе Heroku.
